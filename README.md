# 🎵 MusicChat

**MusicChat**은 인기 차트, 아티스트 및 사용자 기반 검색, 재생목록 기능을 갖춘 음악 탐색 웹 애플리케이션입니다.  
무한 스크롤, 다크 모드, 실시간 미리보기 등 편리한 기능을 통해  
사용자가 원하는 음악을 쉽게 찾고 관리할 수 있도록 돕습니다.

## ✨ 주요 기능

- 🎧 **인기차트**   
  인기 음악을 끌어올 수 있는 차트 기능을 제공합니다.  
  무한 스크롤을 통해 더 많은 곡을 탐색할 수 있도록 구현했습니다.

- 🎤 **아티스트 검색**   
  아티스트의 이름으로 검색하여 해당 아티스트의 대표곡과 관련 정보를 확인합니다.

- 👤 **사용자 검색**   
   MusicChat을 사용하는 다른 사용자의 재생 데이터를 기반으로,  
  자주 듣는 음악을 확인할 수 있습니다.

- 📃 **재생목록** 
  사용자가 선택한 곡을 재생 목록에 추가하거나 삭제할 수 있으며,  
  하단에는 마지막으로 선택한 곡이 미리보기로 표시됩니다.

- **다크 모드** 
  사용자의 환경에 따라 다크 모드로 전환할 수 있도록 구현되어  
  눈에 부담 없이 야간에도 편리하게 사용할 수 있습니다
